# Requirements
- Cmake
- Boost
- Ruby gem (for gherkin with the wire protocol)
    - cucumber
- Cppspec
- Google mock (included in package)
    - https://github.com/google/googletest
- Cucumber-Cpp (included in package)
    - https://github.com/paoloambrosio/cucumber-cpp
